int64_t arcsdec::v_1_0_0::details::wavpack::WavpackOpenFile::read_pcm_samples(const int64_t pcm_samples_to_read, std::vector<int32_t>& buffer) const

Read the specified number of 32 bit PCM samples.

Parameters
pcm_samples_to_read in Number of 32 bit samples to read
buffer in/out The buffer to read the samples to
Returns Number of 32 bit PCM samples actually read
Exceptions
invalid_argument If buffer.size() < (pcm_samples_to_read * 2)

Note that each 16 bit sample will be stored as a single int32, so buffer will require a size of pcm_samples_to_read * number_of_channels to store the requested number of samples.
