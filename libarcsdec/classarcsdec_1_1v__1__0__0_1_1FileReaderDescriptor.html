arcsdec::v_1_0_0::FileReaderDescriptor class
#include <descriptors.hpp>

Abstract base class for the properties of a FileReader.

Contents

  • Reference
    • Derived classes
    • Constructors, destructors, conversion operators
    • Public functions
    • Private functions

A FileReaderDescriptor provides all required information to decide whether a a given file can be read by readers conforming to this descriptor. It can create an opaque reader that can read the file.

FileReaderDescriptors are supposed to be stateless.

Note

Instances of this class are non-copyable and non-movable but provide protected special members for copy and move that can be used in subclasses.
