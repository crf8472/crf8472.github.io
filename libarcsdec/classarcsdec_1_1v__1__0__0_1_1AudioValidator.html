arcsdec::v_1_0_0::AudioValidator class
#include <audioreader.hpp>

Base class for validation handlers for AudioReaderImpls.

Contents

  • Reference
    • Derived classes

    • Constructors, destructors, conversion operators

Implements a class that just provides some assert methods that get a label, a current value, a proper value and an error message. The validating handler keeps an error list and can also return the latest error or the complete list of errors. Subclasses may decide to throw exceptions by implementing on_failure().

It also provides default assertions for validating against CDDA, thereby delegating to CDDAValidator.

Subclasses must implement do_codecs() to provide the list of supported audio codecs, i.e. codecs that are actively validated. If the subclass does not validate any aspects of the codec, it should return an empty set.

Note

Instances of subclasses are non-copyable but movable.
