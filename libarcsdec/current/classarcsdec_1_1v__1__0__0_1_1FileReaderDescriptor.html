arcsdec::v_1_0_0::FileReaderDescriptor class
#include <descriptor.hpp>

Abstract base class for the properties of a FileReader.

Contents

  • Reference
    • Derived classes
    • Constructors, destructors, conversion operators
    • Public functions

A FileReaderDescriptor provides all required information to decide whether a a given file can be read by readers specified and created by this descriptor. It can create an opaque reader that can read the successfully matched file.

Warning

FileReaderDescriptors are supposed to be stateless. If a subclass adds a state, the equality operator will not work as expected.

bool arcsdec::v_1_0_0::FileReaderDescriptor::accepts(const Codec codec) const

Check for acceptance of the specified codec.

Parameters
codec in The codec to check for
Returns TRUE if codec is accepted, otherwise FALSE

A descriptor accepts a Codec iff it is contained in codecs().

Warning

This does not entail that accepts(f, codec) is TRUE for every accepted Format f! The set of formats for which this codec is accepted may be restricted to a subset of the accepted formats().
