arcsdec::v_1_0_0::details::flac::FlacAudioReaderImpl class final
#include <readerflac_details.hpp>

File reader implementation for files in CDDA/Flac format, i.e. containing 44.100 Hz/16 bit Stereo PCM data samples.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, conversion operators
    • Public functions

This class provides the PCM sample data as a succession of blocks of 32 bit PCM samples to its Calculation. The first block starts with the very first PCM sample in the file. The streaminfo metadata block is validated to conform to CDDA.
