arcsdec::v_1_0_0::AudioReader class
#include <audioreader.hpp>

Opaque instance to read audio files and provide the decoded samples.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, conversion operators
    • Public functions

A AudioReader can process an audio file and return its checksums including the ARCSs v1 and v2 for all tracks.
