arcsdec::v_1_0_0::ByteConverter class
#include <audioreader.hpp>

Service class providing methods to convert short byte sequences to integers.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

ByteConverter can be used to parse integers from an input stream of bytes. It provides methods to interpret short sequences of 2 or 4 bytes as a 16 bit or 32 bit integer number.
