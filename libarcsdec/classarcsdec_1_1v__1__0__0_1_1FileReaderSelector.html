arcsdec::v_1_0_0::FileReaderSelector class
#include <descriptors.hpp>

A selection mechanism for a FileReaderSelection.

Contents

  • Reference

A FileReaderSelector applies FileTests to FileReaderDescriptors to select a descriptor with a certain test result.

It implements two different decisions. Implementing matches() defines which descriptors are candidates to be selected. Implementing select() defines which of the matching candidates is concretely selected.
