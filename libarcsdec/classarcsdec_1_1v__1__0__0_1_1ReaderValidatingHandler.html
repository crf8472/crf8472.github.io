arcsdec::v_1_0_0::ReaderValidatingHandler class
#include <audioreader.hpp>

Base class for validation handlers for AudioReaderImpls.

Contents

  • Reference
    • Derived classes
    • Constructors, destructors, conversion operators
    • Public functions
    • Protected functions

Implements a class that just provides some assert methods that get a label, a current value, a proper value and an error message. The validating handler keeps an error list and can also return the latest error or the complete list of errors. Subclasses may decide to throw exceptions accordingly.
