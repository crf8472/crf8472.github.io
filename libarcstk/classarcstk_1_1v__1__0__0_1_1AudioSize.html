arcstk::v_1_0_0::AudioSize class final
#include <calculate.hpp>

Uniform access to the size of the input audio information.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

Some decoders provide the number of frames, other the number of samples and maybe in some situations just the number of bytes of the sample stream is known. To avoid implementing the appropriate conversion for each decoder, AudioSize provides an interface for uniform representation to this information. Any of the informations provided will determine all of the others.
