arcstk::v_1_0_0::DefaultContentHandler class final
#include <parse.hpp>

Constructs an ARResponse instance from parsed content.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, conversion operators

Create an ARResponse and use set_object() to inform the handler about that object. The handler instance will then use this object for construction. After parse() is finished, the object contains the parsed content.
