arcstk::v_1_0_0::TracksetMatcher class final
#include <match.hpp>

Match an arbitrary set of tracks against an ARResponse.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, conversion operators
    • Public functions

Find any match of any actual Checksum in the ARResponse. This targets the situation where a subset of tracks from the same album are tried to be matched, but the subset may be incomplete.

Example:

You have some or all tracks but you do not know their order. You can use the TracksetMatcher to find out the order of the tracks.
