arcstk::v_1_0_0::AlbumMatcher class final
#include <match.hpp>

Match an album track list against an ARResponse.

Contents

  • Reference

Tries to match each position i in the actual Checksums with position i in each ARBlock of the ARResponse. This is how an entire album can be matched.
